## Bittorrent simples

### Objetivo
  Aprender os conceitos básicos de um cliente de torrent e no processo reforçar os meus conhecimentos de JavaScript.

### Material de estudo
  Pelo repositório [codecrafters-io](https://github.com/codecrafters-io/build-your-own-x) eu encontrei esse [artigo](https://allenkim67.github.io/programming/2016/05/04/how-to-make-your-own-bittorrent-client.html) que passa pelo processo de desenvolvimento de um cliente bittorrent simples em JavaScript. Vi uma excelente oportunidade para aprender os conceitos de uma ferramenta com uma forte presença na internet, e de quebra aprender ainda mais sobre JavaScript. Aqui agradeço o [Allen Kim](https://github.com/allenkim67) por produzir o material.
